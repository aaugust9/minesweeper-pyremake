# minesweeper-pyremake
Remake of Microsoft's Minesweeper 1990 with a view extra styles to choose from.

### Description.
Small project to learn about pygame and what it can do. A style menu was added to the options tab incase you wanted to be "true to classic", you could change the look of your game!
Code is barebones and needs refining to be release quality.

### Starting.
Run this command inside the root folder to run and play the game 
`python3 main.py`
